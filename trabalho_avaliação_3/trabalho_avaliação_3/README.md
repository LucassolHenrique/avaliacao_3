Como iniciar a interface gráfica do projeto:

1. Certifique-se de ter o Python instalado na máquina.
2. Abra a pasta do projeto no terminal/prompt de comando.
3. Digite e execute o comando:
python interface_avl.py

Arquivos:
- avl_exclusao.py: Contém o código da árvore AVL (inserção, rotações e remoção).
- interface_avl.py: Contém a interface do visualizador gráfico (feita em Tkinter).
- passo_a_passo_ppt.txt: Contém o roteiro resumido/explicado para a criação dos slides do PowerPoint.
